### User Stories
#### Pet-owner perspective:
- Chinois
  - 我可以注册一个新帐户并选择我的角色，这样我就可以登录了
  - 我可以登录我的账户，这样我就可以使用市场了
  - 我可以看到我发布的职位列表
  - 如果我发布了一个职位，我可以查看并修改其细节
  - 如果我发布了一个职位，我可以删除它.
  - 如果我发布了一个职位，我可以看到申请的宠物看护者
  - 如果我找到了合适候选人，我可以批准他们
  - 我可以修改我的账户信息.
  - 我可以删除我的账户.
